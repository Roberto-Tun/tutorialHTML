HTML - Exponente

Para producir un texto superscript usaremos la etiqueta <sup>.

HTML 
<p>Este texto es<sup>exponencial!</sup></p>

HTML - Exponentes

Podemos usar la etiqueta <sup> para escribir expresiones matemáticas, como puedes ver:

HTML 
3<sup>2</sup> = 9
<br />
14<sup>x</sup>

HTML - Notas de pie de página

El superscript también puede ser usado para referencias, explicaciones, y cualquier otra información adicional en un pie de página.

HTML 
<p>Una simple frase para evidenciar el exponente<sup>1</sup> en HTML.</p>
<hr />
<p><small>1. El exponente de un número nos dice cuántas veces se usa el número en una multiplicación.</small></p>

Puedes practicar con este elemento, que se utiliza raramente pero es bastante útil.
