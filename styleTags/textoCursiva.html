HTML - Texto cursiva

Las etiquetas para poner el texto en cursiva, al igual que la etiqueta italic, se usa para subrayar ciertos términos o palabras claves, a veces una frase entera. Aunque, no es recomendable abusar de su uso del texto en cursiva.

Para crear este tipo de texto podemos usar lo siguiente:

HTML
<i>Italic</i>. Texto estilo cursiva!<br />
Etiqueta <em>Emphasized</em>. Texto resaltado, normalmente en cursiva. Dependiendo del dispositivo puede ser resaltado de otras formas!

Demo

Italic. Texto estilo cursiva!
Etiqueta Emphasized. Texto resaltado, normalmente en cursiva. Dependiendo del dispositivo puede ser resaltado de otras formas!

Normalmente, el navegador interpretará estas etiquetas de la misma manera.

Por lo general, la más usada es em, dado que puede ser interpretada según las necesidades en el dispositivo o aplicación donde aparece. Si se necesita simplemente un texto con estilo cursiva sin importar lo demás, se puede usar CSS en lugar de la etiqueta italic.

HTML
<!-- Usando las etiquetas de estilo -->
<p>
<b>HTML</b><br />
<i>Hyper Text Markup Language</i>
</p>

<!-- O usando etiquetas semánticas -->

<p>
<strong>HTML</strong><br />
<em>Hyper Text Markup Language</em>
</p>

<!-- O usando CSS inline-->

<p>
<span style="font-weight:bold">HTML</span><br />
<span style="font-style:italic">Hyper Text Markup Language</span>
</p>

La mayoría de las veces, negrita y cursiva se ponen dentro de otras etiquetas para obtener el formato final del texto (como por ejemplo en un parágrafo, o span o div). Dicho esto, bajo ninguna circunstancia se debe olvidar cerrar una etiqueta y además en el mismo orden de apertura.

Todas las etiquetas deben ser cerradas, en el mismo orden en el que se han abierto
