HTML - Etiqueta code

La etiqueta <code> te permite darle un formato específico al texto como si fuera código de ordenador. Esto ofrece un determinado tamaño y una distancia específica determinada para el código del ordenador.

HTML 
<code>Este texto fue formateado con la etiqueta code.</code>
