HTML - Texto en negrita - aplicaciones

Generalmente, el texto en negrita, es usado para enfatizar ciertas palabras o términos dentro de una oración.

HTML 
<p><b>El texto</b> en negrita es útil para enfatizar ciertos<b>términos</b></p>

HTML - Etiqueta strong

La etiqueta strong resalta un texto de manera semántica. Normalmente se muestra en negrita, pero tiene un comportamiento diferente en varios dispositivos.

HTML 
<p>Este es una <b>palabra en negrita</b> y esta otra es una <strong>palabra resaltada</strong> usando la etiqueta<em>strong</em>. Lo más probable es que las dos se muestren muy parecido.</p>

Demo

Este es una palabra en negrita y esta otra es una palabra resaltada usando la etiqueta strong. Lo más probable es que las dos se muestren muy parecido.

Así que para aclarar todo, bold se usa para dar estilo a un elemento HTML, mientras que strong es una etiqueta semántica que resalta una palabra clave según el contexto.
