HTML - Texto tachado

El texto tachado puede hacerse usando la etiqueta <del>.

HTML 
<p>Este texto está <del>cortado</del> por una línea!</p>

Texto tachado - Aplicaciones

Esta etiqueta no tiene muchas aplicaciones, pero intentaremos mostrar su uso en el siguiente ejemplo: una lista de compras.

HTML 
<ol>
    <li>Un iPhone</li>
    <li><del>Queso</del></li>
    <li><del>Leche fresca</del></li>
</ol>
