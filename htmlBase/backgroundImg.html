HTML - Imágen de fondo

HTML > Tutorial HTML - Imágen de fondo

Ademas de un color sólido para el fondo, puedes también usar una imagen. Para hacer esto puedes ver siguiente el ejemplo.

HTML 
<table height="100" width="100"    background="../img/image.jpg">
    <tr>
        <td>Esta tabla tiene una imagen como fondo</td>
    </tr>
</table>

HTML - Fondo repetido (tile)

En el primer ejemplo, tanto la tabla como la imagen tenian el mismo tamaño. Pero hay situaciones donde la tabla es más larga que la imagen, y entonces la imagen se multiplicará a sí misma hasta alcanzar el ancho y alto de la tabla.

HTML - Fondos con patron (pattern)

Repetir una imagen tiene otro uso como vamos a explicar a continuacion. Esta función del navegador de repetir la imagen sin fin puede ser usada como una ventaja. Podemos crear una imagen muy pequeña (con la ayuda de Photoshop o cualquer programa de edicion grafica), por ejemplo de 5x5 pixeles. Esta imagen puede ser un modelo concreto que repetirá infinitamente la imagen y hara un efecto de continuidad

HTML - Color transparente de fondo (transparent)

Otro método de usar la imagen repetida es el fondo transparente. Para conseguir esto se suelen usar imagenes con la extension PNG o GIF semitransparentes. Puedes poner de fondo este tipo de imagenes de la misma forma que cualquer otra image.
