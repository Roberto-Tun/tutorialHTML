HTML - <!--Comentarios--!>

Los comentarios se usa muy a menudo en los códigos HTML. Por eso es muy importante que no estén indicados en el navegador para evitar que los muestre.

Hay muchas razones para agregar un comentario dentro de un script html.

  • Una nota o un recordatorio, una especificación o una aclaración.
  • Comentarios Javascript
  • Un elemento sin terminar

El comentario estará ubicado entre las etiquetas "<!--" y "-->". De esta manera, podremos dejar una nota y así recordar después para qué era el código, o saber sí en ese lugar hay que a;adir algo mas.

HTML 
<!--The beginning of the code that shows a photo-->

<p>Este es un párrafo entre dos líneas de comentarios html</p>

<!--The end of the code that shows a photo-->

Esto es solamente un ejemplo de lo que se puede comentar en un script. Con el tiempo veras que estos comentarios son bastante útiles para mejorar, mas tarde, la lectura del código.

Cualquier texto, letras números o símbolos o cualquier cosa ubicada dentro de la etiqueta de abertura "<!--" y la de cierre "-->" no será mostrado en el navegador.

HTML - Script sin terminar

Los comentarios son muy útiles cuando trabajas en un script, pero lo dejas sin terminar. Colocarlo dentro de las etiquetas (<!-- code -->) será interpretado como un comentario por el navegador y entonces no se mostrara.

HTML 
<!-- <input type="text" size="12" /> -- Input Field -->

Cuando has terminado el código y necesitas mostrarlo todo lo que necesitas hacer es borrar las dos etiquetas de apertura y cierre del comentario y el navegador mostrará el código deseado.

HTML 
<input type="text" size="12" />
