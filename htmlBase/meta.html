HTML - Etiqueta meta

La etiqueta meta es usada en general, para proporcionar información adicional al motor de búsqueda. Esta información no será visible a menos que veas el código fuente de la página HTML.

HTML Meta - Palabras clave

En la etiqueta meta puedes agregar las palabras clave que definen el contenido de la página.

HTML 
<head>
<meta name="keywords" content="Tutorials, HTML, web, resources, webmaster" />
</head>

HTML Meta - Descripción

Esta etiqueta te ayuda a describir tu página web.

HTML 
<head>
<meta name="description" content="A web site resource for those willing to learn HTML and for webmasters." />
</head>

HTML Meta - Revisado

Es usada para especificar la última fecha de edición de la página web.

HTML 
<head>
<meta name="revised" content="1/08/2015" />
</head>

HTML Meta - Actualizar y redireccionar

Para actualizar la página mientras el usuario la está viendo, se debe usar http-equiv="refresh".

HTML 
<head>
<meta http-equiv="refresh" content="120; url=http://www.tutorialehtml.com" />
</head>

La redirección se hace fácilmente cambiando la dirección de la página web con la que desees redirigirla.
