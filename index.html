HTML - Introducción al HTML

HTML > Tutorial HTML > HTML - Introducción al HTML

Bienvenido a la sección de tutoriales HTML. Aquí podrás aprender a utilizar el lenguaje HTML. Después de leer este tutorial podrás hacer tus propias páginas web.

Si ya estás familiarizado con el HTML, entonces será mucho más fácil de aprender. Te recomiendo, que no leas este tutorial de principio a fin sino que utilices un cuarto o media hora de tu tiempo para que practiques después de haber leído varias lecciones.

Antes de empezar a aprender HTML

Crear documentos en HTML no es difícil. Para empezar, sólo necesitas un bloc de notas y dedicación. Si eres nuevo en este ámbito y aun no has leído la Guía para principiantes, te recomiendo que pases unos cuantos minutos haciéndolo.

Páginas web

Las páginas web son bastante útiles. Esto explica algunos argumentos:

  • La manera más fácil de propagar información en Internet
  • Otra forma de expandir tu negocio
  • Puedes mostrarle al mundo lo que tienes que decir en una página personal

Términos a recordar

Etiqueta - utilizada para especificar las regiones del documento HTML, que luego serán leídas por el navegador. Las etiquetas se verán así:

Elemento - es una etiqueta completa, teniendo una <etiqueta> de apertura y una etiqueta de cierre </etiqueta>.

Atributo - se usa para cambiar el valor de un elemento HTML. Normalmente un elemento tiene varios atributos.

Hasta ahora, recuerda solamente que una etiqueta es un comando que el navegador ejecuta, un elemento es una etiqueta completa y un atributo personaliza o de estilos diferentes a un elemento HTML.
