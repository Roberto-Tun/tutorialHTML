HTML - Etiqueta body

Inicio »   HTML »   Tutorial HTML - Etiqueta body

Como mencionamos antes en otro tutorial, la etiqueta <body> es la que contiene cada elemento visible en la página web: tablas, marcos, formularios, párrafos, fotos y demás.

Todos estos se ubican entre la etiqueta <body> y </body>.

HTML - Atributos de la etiqueta body

  • leftmargin - establece un borde en el lado izquierdo de la página (un espacio vacío)
  • topmargin - establece un borde en la parte superior de la página (un espacio vacío)

Un ejemplo sería:

HTML
<body topmargin="50">

<p>Este texto está ubicado a una distancia de 50 pixeles de la parte superior de la página.</p>
</body>

HTML
<body leftmargin="50">

<p>Este texto está ubicado a una distancia de 50 pixeles del lado izquierdo de la página</p>
</body>

Puedes copiar este código en un archivo de texto y guardarlo como .html para verlo mejor.

HTML
<body text="red">

o

<body text="rgb(255,0,0)">

Esta parte del código establecerá el color base de la fuente, como color rojo, en caso de que otro color no sea especificado dentro de la etiqueta de párrafo, <p>.

HTML - Color del link establecido por la etiqueta body

Igual que con el texto, puedes poner el color de los links vistos o no vistos con la ayuda de la etiqueta body.

HTML
<body link="white" vlink="black">

o

<body link="rgb(255,255,255)" vlink="rgb(0,0,0)">

En realidad esto se usa muy poco, dado que en su lugar se suele usar el CSS (Cascading Style Sheets), es mucho más fácil y además práctico para futuros modificaciones.
