HTML - Símbolos

HTML > Tutorial HTML - Etiquetas

En HTML, entity es un nombre de adorno para los símbolos. Algunos símbolos como por ejemplo copyright ( © ), trademark ( ™ ) y muchos otros, están disponibles en el teclado, pero en HTML necesitan ser escritos de una manera especial.

  • Empieza con el signo - &
  • Después de eso escribiremos el nombre del signo - copy
  • Al final del - ;

Símbolo del copyright

Usa © para hacer - © - el símbolo del Copyright

Espacios múltiples y < >

Como dije en la lección sobre párrafos, un navegador reconocerá un espacio únicamente, sin importar cuantos de estos uses en un código HTML. Pero hay una entidad o símbolo que puede ser introducido para mostrar más de un espacio.

HTML 
<p>En esta frase se han introducido &nbsp;&nbsp;&nbsp;&nbsp; algunos espacios adicionales.</p>


Demo

En esta frase se han introducido      algunos espacios adicionales.

Para mostrar códigos HTML tienes que usar los símbolos de "mas" y "menos". Para que sean mostrados en el navegador debes usar un símbolo.

HTML 

<p>
    Less than - &lt; <br />
    More than - &gt; <br />
    Head tag - &lt;head&gt;
</p>


Demo

Less than - <

More than - >

Head tag - <head>
