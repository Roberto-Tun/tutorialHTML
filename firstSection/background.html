HTML - Bgcolor

HTML > Tutorial HTML - Bgcolor

El atributo bgcolor es usado para determinar el color de fondo de un párrafo, tabla o cualquier otra parte del HTML. Es recomendable el uso moderado de esta etiqueta. También es recomendable establecer los colores de fondo y los estilos de texto o links, con la ayuda de CSS. Explicaremos con mas detalles cómo puede hacerse eso en un tutorial CSS posterior.

<"tag" bgcolor="Value">

Este es un modelo de cómo el atributo bgcolor sería utilizado dentro de una etiqueta.

HTML 
<body bgcolor="Silver">

HTML - Agregar color a las tablas

El siguiente ejemplo te muestra cómo hacer un efecto zebra en una tabla, usando el atributo bgcolor.

HTML 
<table>

    <tr bgcolor="#FFFF00">
        <td>Esta Fila es Amarilla!</td>
    </tr>

    <tr bgcolor="#AAAAAA">
        <td>Esta Fila es Gris!</td>
    </tr>

HTML - Fuente y fondo juntos

Pondremos en práctica un ejemplo de una "tabla resultados", coloreando la tabla para destacar ciertas secciones, según lo deseado.

HTML 
<table>

    <tr>
        <td bgcolor="#009900"><font color="#FFFF00">Green Bay</font></td>
        <td bgcolor="#000000"><font color="#FFFFFF">27</font></td>
    </tr>

    <tr>
        <td bgcolor="#0000FF"><font color="#DDDDDD">New England</font></td>
        <td bgcolor="#000000"><font color="#FFFFFF">20</font></td>
    </tr>

</table>

Un ejemplo de un párrafo con un fondo gris:

HTML 
<table>
    <tr bgcolor="#777777">
        <td>
            <font face="Monotype Corsiva, Verdana"size="4" color="#00FF00">
            Esta etiqueta de párrafo tiene fondo gris con la fuente de color verde. 
            Usted debe ver la fuente Monotype Corsiva si lo tiene instalado, 
            o Verdana como la copia de seguridad. 
            Ambas fuentes son ampliamente aceptados como fuentes estándar.
            </font>
        </td>
    </tr>
</table>


Demo

Esta etiqueta de párrafo tiene fondo gris con la fuente de color verde. Usted debe ver la fuente Monotype Corsiva si lo tiene instalado, o Verdana como la copia de seguridad. Ambas fuentes son ampliamente aceptados como fuentes estándar.
