HTML - Párrafo

HTML > Tutorial HTML - Atributos

El párrafo es un elemento base en la edición de texto. La etiqueta para un párrafo es <p>.

Este atributo deja un espacio vacío encima y debajo del texto para evidenciarlo, así tanto el navegador como el usuario lo podrá distinguir como tal.
