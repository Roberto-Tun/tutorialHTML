HTML - Envío del formulario

HOME » HTML » Tutorial HTML - Envío del formulario

Los botones submit son otro uso de la etiqueta <input>. Estos muestran un botón típico, el cual ejecutará la acción de enviar el formulario.

HTML 
<input type="submit" value"Submit"/>
<br>
<input type="submit" value="Enviar"/>
<br>
<input type="submit" value="Continuar"/>

Demo

Como puedes ver hicimos una variación de botones de envío, cambiando el valor del atributo. Esto puede ser modificado dependiendo de las necesidades.

Submit - El envio

Para que un botón submit funcione correctamente es necesario poner la etiqueta <form> además de otros pocos atributos. Esos son method y action. Además necesitaremos un archivo PHP, PERL o ASP, para complementar esta acción.

Otra opción sería poner el valor del atributo action como mailto, seguido de una dirección e-mail, y el formulario será enviado a la dirección e-mail.

HTML 
<form method="post" action="mailto:email@exemple.com" >
Nombre: <input input="text" name="Name" size="15" maxlength="15" />
Apellido: <input input="text" name="Last_name" size="25" maxlength="15" />
<input type="submit" value="Enviar por correo" />
</form>
