HTML - Campo contraseña

  »   HTML  »   Tutorial HTML - Campo contraseña

Los campos de la contraseña son una categoría especial de la etiqueta input. Aun así, escribirlos es tan simple como escribir un campo de texto normal.

HTML 
<input type="password" size="5" maxlength="5" />
<input type="password" size="10" maxlength="10" />

Demo

Y el resultado es un campo en el cual el texto está escondido en forma de asteriscos, estrellas o círculos negros, esto depende de la interpretación del navegador.

Campos de contraseña en HTML - Nota

Como puedes ver, no hay una gran diferencia entre la estructura de un campo de texto normal y la estructura de un campo que contiene una contraseña.

Debe decirse que estos campos de contraseña no están encriptados, y usarlos sin ninguna medida de seguridad, tal y como se muestran arriba no es recomendable.

El envío de contraseñas cifradas con el servidor se puede hacer usando SSL / HTTPS. Otra forma sería utilizar .htaccess (si estás usando Apache). Pero estos son temas técnicos que aprenderás más adelante.
