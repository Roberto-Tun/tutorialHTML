HTML - Campos de texto

  »   HTML  »   Tutorial HTML - Campos de texto

Los campos de texto se encuentran en los formularios. Estos, son procesados con la ayuda de un lenguaje de programación, la mayoría de las veces ASP, PERL o PHP.

Atributos adicionales que son utilizados para crear los campos de texto son:

  • size
  • value
  • maxlength

Daremos un ejemplo de cada uno de ellos:

HTML - El tamaño o el ancho del campo de texto

El atributo size establece la longitud del campo de texto. La longitud estándar de un campo de texto está por lo general entre los 20-25 caracteres. Pero esto puede variar dependiendo del propósito del formulario o del campo de texto en sí.

HTML 
                    
<input type="text" size="5" />
<br />
<input type="text" size="15" />
<br />
<input type="text" size="25" />

Demo

HTML - Otro atributo es el value (valor)

Usando este atributo podremos asignar un valor por defecto a nuestros campos, información que el usuario puede dejar o borrar. Esta aplicación es bastante útil, especialmente cuando un script PHP está integrado para reconocer un visitante y mantiene las opciones elegidas por el usuario.

HTML 
<input type="text" size="5" value="12345" />
<br />
<input type="text" size="15" value="HTML text field" />
<br />
<input type="text" size="25" value="HTML Tutorials" />

Demo

HTML - Maxlength

Maxlength es uno de los atributos más útiles para los campos de texto, ya que este no le permite al usuario teclear un número de caracteres mayor al permitido.

En caso de que quieras introducir más de 5, 15 y 25 caracteres en los campos de arriba, esto no será posible debido a la restricción aplicada por 'maxlength'.
