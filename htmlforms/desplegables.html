HTML - Desplegables

Las listas drop down (listas desplegables) son uno de los tipos de listas más prácticas. Probablemente las hayas visto muchas veces en Internet, pero sin saber que tenían ese nombre.

HTML 
<select>

    <option>New York</option>
    <option>Bucharest</option>
    <option>Madrid</option>

</select>

Demo

El navegador mostrará automáticamente la primera opción. Pero, esto puede cambiarse con la ayuda del atributo "selected".

HTML 
<select name="my_html_select_box">

    <option>New York</option>
    <option selected="yes">Bucharest</option>
    <option>Madrid</option>

</select>

Demo

HTML - Formularios de selección

Utilizaremos el atributo size para cambiar la altura de una lista drop down en un formulario de selección.

HTML 
<select name="my_html_select_box" size="3">

    <option>New York</option>
    <option>Bucharest</option>
    <option>Madrid</option>

</select>

Demo

HTML - Selecciones múltiples

En caso de que el usuario desee seleccionar más de uno de los enviados, será mucho más fácil con la ayuda del atributo multiple.

HTML 
<select name="my_html_select_box" multiple="yes" size="3">

    <option>New York</option>
    <option>Bucharest</option>
    <option>Madrid</option>

</select>

Demo

Se deduce que este atributo multiple no funcione con una lista drop down y solo funciona cuando el atributo size está especificado.
