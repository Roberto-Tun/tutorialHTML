HTML - Campos-escondidos

Los campos escondidos no serán mostrados por el navegador. Estos son necesarios cuando tenemos que tratar con formularios y bases de datos en MySQL, pero no se limita a eso. Utilizaremos los campos escondidos para enviar información adicional a la base de datos, además de la información enviada por el usuario.

HTML 
<input type="hidden" />

Este fragmento de un código no mostrará nada porque el navegador lo trata como información que no debe ser mostrada.

HTML - Otros atributos

Utilizaremos los atributos name para establecer un nombre a nuestro campo escondido. Podemos usar el atributo id por si necesitamos referenciar el campo desde un lenguaje como Javascript.

HTML 
<input type="hidden" id="age" name="age" value="25" />

<input type="hidden" id="DOB" name="DOB" value="01/01/70" />

<input type="hidden" id="admin" name="admin" value="1" />

Arriba mostramos los ejemplos de tres modelos de campos escondidos que pueden ser usados especialmente si tienes una página web donde el usuario tiene que hacer login para acceder a determinada información. El campo de 'admin' es utilizado para ver el rango de alguien, 1 siendo administrador, y 0 no-administrador.

Los 'Hidden fields' son usados cuando tenemos que tratar con información que queremos dejar en más de un formulario, para así no obligar al usuario a insertar la información varias veces.
