HTML - Formulario de upload

Inicio » HTML » Tutorial HTML - Formulario de upload

El campo upload es un campo muy práctico dentro de un formulario, permitiendo a los usuarios subir fotos, documentos y cualquier clase de archivos al servidor.

Para crear un formulario de subida de archivos, sólo tenemos que establecer el valor file en la etiqueta <input type="">.

HTML 
<input type="file" />

Demo

Upload - Tamaño máximo de archivo

Para limitar el tamaño de algunos de los archivos subidos en el servidor web usaremos el campo hidden. Esto no quiere decir que el archivo, no se volverá a chequear por parte del servidor.

HTML 
                    
<input type="hidden" name="MAX_FILE_SIZE" value="4194304" />
<input type="file" />

El valor escogido en el ejemplo de arriba fue 4194304. Esto significa que los archivos de más de 4194304 bytes (4MB) no se podrán subir.
