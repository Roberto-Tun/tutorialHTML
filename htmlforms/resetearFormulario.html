HTML - Resetear formulario

Los botones reset son usados con frecuencia en formularios grandes, en caso de que el usuario haya escrito mal la mayoría de la información, y desea borrar todo el texto introducido.

HTML 
<input type="reset" value="Reset" />

<input type="reset" value="Borrar" />

<input type="reset" value="Limpiar formulario" />

Demo

HTML - El botón reset

Para que el botón reset sea funcional, será necesario que esté dentro de la etiqueta <form>. Puedes leer el tutorial sobre formularios HTML para más ejemplos.

HTML 
<form action="myphp.php" method="post">

    <input type="text" size="15" maxlength="15" />

    <input type="text" size="25" maxlength="25" />

    <input type="reset" value="Borrar todo" />

</form>

Demo

Intenta insertar un pequeño texto y luego dale click al botón "Borrar todo" en el ejemplo de arriba.
